# 📊 CodeAlpha_SalesAnalysis

## 📌 Project Overview
This project focuses on analyzing sales data and transforming it into meaningful visualizations. It helps in understanding trends, comparing categories, and evaluating regional performance.

---

## 📂 Dataset
The dataset includes the following columns:
- Date – transaction date  
- Sales – revenue generated  
- Category – product category (Technology, Furniture, Office Supplies)  
- Region – geographical region (West, East, North, South)  

---

## 📊 Visualizations
- 📈 Sales Trend Over Time  
- 📊 Sales by Category  
- 🌍 Sales by Region  

---

## 🔍 Key Insights
- Technology category has the highest sales  
- West region performs better compared to others  
- Sales show an increasing trend over time  

---

## 🛠 Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## 🚀 How to Run
1. Install required libraries:
   pip install pandas matplotlib seaborn  

2. Run the project:
   python visualization.py  

---

## 🎯 Conclusion
This project demonstrates how data visualization can be used to extract insights from raw data and support decision-making.
